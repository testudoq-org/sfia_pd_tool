SFIA PD Generator

Instructions

Welcome to the PD generator tool, built upon the SFIA 8 Framework.

To initiate PD generation, follow these steps:

  1. Select Skill Set Levels: Choose the desired skill set levels from the SFIA Information Table. Each skill level corresponds to a specific competency level.
  2. Hover for Descriptions: For more information about any skill, hover over the skill name. Additionally, hover over the checkboxes next to each skill level to see detailed descriptions of what each level entails.
  3. Review Your Selections: Your selected choices will be formatted and displayed at the bottom of the page. You can review them before exporting.
  4. Export Options: You can export your selections as a CSV or HTML document. Click the respective button to download the file automatically. Ensure you are using an up-to-date version of Chrome or Firefox for the best experience, as some browsers may have limitations.
  5. Bookmark for Future Reference: To save your summary for future reference, bookmark this page. The URL bookmark retains the necessary information and will update your selections upon your return.

Additional Tips:

  • Make sure to check all relevant skill levels for a comprehensive position description.
  • If you encounter any issues, try refreshing the page or clearing your browser's cache.
  • For optimal performance, consider using a desktop or laptop device, especially when working with larger datasets.

If you have any questions or need further assistance, please refer to the help section or contact support.
